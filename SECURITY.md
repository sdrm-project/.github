# Security Policy

The SDRM project takes security seriously.

If you discover a security vulnerability in any SDRM repository, please report it responsibly.

## Reporting a Vulnerability

Please report security issues privately by contacting:

sdrm.systems@gmail.com

Include the following information if possible:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested mitigation if known

## Responsible Disclosure

Please do not publicly disclose security vulnerabilities until they have been reviewed and addressed by project maintainers.

This helps protect users and infrastructure that may rely on SDRM components.
